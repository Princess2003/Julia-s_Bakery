# 🍰 Julia's Bakery Website

## 📌 Project Overview
Julia's Bakery Website is a simple and visually appealing website created to showcase a bakery's products, services, and contact information. The website highlights delicious baked goods such as cakes, cookies, muffins, and bread while providing visitors with information about the bakery and how to get involved.

This project demonstrates basic **front-end web development skills** using HTML and CSS to create a structured multi-page website.

---

## 🌐 Website Pages
The website includes the following pages:

- **HomePage.html** – The main landing page introducing the bakery.
- **ABOUT US.html** – Information about the bakery and its story.
- **OUR WORK.html** – Displays the bakery's creations and services.
- **PHOTO GALLERY.html** – A gallery of baked goods and bakery images.
- **GET INVOLVED.html** – Information about participating or supporting the bakery.
- **CONTACT US.html** – Contact details for customers.

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website
- **CSS3** – Styling and layout
- **JavaScript Tools**
  - Node.js
  - Gulp
  - Grunt
- **Bower** – Frontend package management

## 🚀 How to Run the Website

1. Download or clone the repository:

```
git clone https://github.com/yourusername/julias-bakery-website.git
```

2. Open the project folder.

3. Navigate to the `public_html` folder.

4. Open **HomePage.html** in your web browser.
